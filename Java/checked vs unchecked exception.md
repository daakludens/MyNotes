# Checked Exception vs Unchecked Exception
### checked exception
- 컴파일 때 실행
- 프로그램의 외적 영향으로 생김
- throws 키워드나 try-catch 문을 사용해 예외 처리 가능
- 사용자가 해당 예외로 부터 복구할 수 잇다
- 예시 : IOException, SQLException

<br>
